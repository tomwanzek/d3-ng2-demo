Collapsible tree

Demonstrate how to:

  • build an Angular 2 component which uses a D3 service to render to <svg>, and
  • use D3 version 4 in TypeScript to draw a collapsible hierarchical tree.

Reference

This example is based on Collapsible tree diagram in v4.

D3 Angular 2 Implementation

The D3 component can be embedded into a parent component using the following tag:

<app-collapsible-tree [data]="someData" ></app-collapsible-tree>.
